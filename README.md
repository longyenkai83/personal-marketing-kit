## 📦 Install — 5 phút (Vietnamese)

### Bước 1: Cài Claude Code (nếu chưa có)

Download: [claude.com/claude-code](https://claude.com/claude-code)

### Bước 2: Tải kit về máy

**Cách nhanh nhất — Download ZIP:**

1. Vào https://github.com/longyenkai83/personal-marketing-kit
2. Click nút xanh **"Code"** → **"Download ZIP"**
3. Giải nén ZIP

### Bước 3: Drop vào `~/.claude/`

| OS | Vị trí |
|----|--------|
| Windows | `C:\Users\[YourUsername]\.claude\` |
| Mac | `/Users/[YourUsername]/.claude/` |
| Linux | `/home/[YourUsername]/.claude/` |

Drop:
- Toàn bộ folder `skills/*` → `~/.claude/skills/`
- File `CLAUDE.md` → `~/.claude/CLAUDE.md`
- Folder `shared/` → `~/.claude/shared/` (optional cho cross-skill)

### Bước 4: Restart Claude Code + verify

Đóng Claude Code hoàn toàn → mở lại → gõ:

```
Tôi có những skill nào trong kit?
```

✅ **PASS** nếu Claude liệt kê được 10 skill (fb-insight-miner, kallaway-hook-master, etc).

## 🎯 Key Methodologies (built-in)

### 🟦 Maria Wendt System ($18M revenue)
- **3-Pillar Viral Product**: Pique Curiosity + Solve Specific Problem + Promise Unbelievable
- **Email funnel**: campaigns / launches / automations (1-2 email/day frequency)
- **Subject line**: 50% time, random + curiosity, viết TRƯỚC body
- **9-Word Email** (Dean Jackson): re-activate cold list trong 9 từ
- **Algorithm hack**: look-like-personal email + reply-back tactic
- **Pricing**: sweet spot $50-100, raise after 100+ sales
- **Tech stack**: Kajabi + Sam Cart only

### 🟢 Kallaway System (viral video)
- **6 Hook frameworks**: Fortune Teller / Experimenter / Teacher / Magician / Investigator / Contrarian
- **5 Dream Outcome formats**: About Me / If I / To You / Can You / He-She Just Did
- **4-Component Alignment**: Spoken + Text + Visual + Audio (Max Alignment)
- **6 Levels Storytelling**: Reporter (avoid) → Trickster → Warrior → Architect → Translator → Artist
- **6 Story Locks**: Naming / Embedded Truths / Thought Articulation / Negative Frames / Loop Openers / Contrast Words
- **Underlying mechanism**: CONTRAST

### 💰 Tripwire Funnel ($50k/month playbook)
- $1-7 entry product (foot-in-door psychology)
- Welcome Coupon: 4 emails over 4 days (immediate / day 1 / day 2 urgency / day 3 last chance)
- Post-purchase: 5 emails over 7 days driving mid-tier
- One-click upsell at thank-you page

### 🎯 9 CTA Library (Diversified)
**Anti-pattern**: Mọi reel/email = "Comment X nhận DM Y" → audience burnout
**Solution**: 70/20/10 rotation
- 70% Educational + Reflection + Save
- 20% Discussion + Share + Action
- 10% Lead-gen + Follow
