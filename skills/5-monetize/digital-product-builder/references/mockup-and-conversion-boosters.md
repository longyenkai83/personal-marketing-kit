## 🎯 Tại sao Mockup CRITICAL

### Digital product khác physical product

| Physical product | Digital product |
|------------------|-----------------|
| Audience touch, see, weigh | Audience CHỈ thấy mockup |
| Photos hiển thị quality | Mockup phải SHOW value visually |
| Returns easy if not satisfied | Refund harder, perception matters more |

→ Mockup = **single biggest conversion lever** cho digital products.

### Math impact

- **Bad mockup**: 100 visitors → 20 checkout views → 2 sales (2% conversion)
- **Good mockup** (Maria style): 100 visitors → 60 checkout views → 12 sales (12% conversion)

→ Same product, mockup quality = **6x revenue**.

### Visual structure

```
┌────────────────────────────────────┐
│                                    │
│  [BIG PRODUCT IMAGE]               │
│  - Course: laptop screen + phone   │
│  - PDF: stack of pages             │
│  - Audio: AirPods + waveform       │
│                                    │
│  + [SMALLER ICONS OF EXTRAS]       │
│    + workbook                      │
│    + bonus PDF                     │
│    + checklist                     │
│                                    │
│  ╔══════════════════════╗          │
│  ║  100+ TEMPLATES      ║  ← BIG #│
│  ╚══════════════════════╝          │
│                                    │
│  ✓ [Value bullet 1]                │
│  ✓ [Value bullet 2]                │
│  ✓ [Value bullet 3]                │
│  ✓ [Value bullet 4]                │
│                                    │
│  + [SUBTLE "+" SIGN]               │
│    "...and more bonuses inside"    │
│                                    │
└────────────────────────────────────┘
```

### 3 GOLDEN RULES

#### Rule 1: Show ALL bundled items

Don't hide what's included. Show EVERYTHING + use "+" sign for any extras.

✅ Good: Image of course + workbook + checklist + bonus → "+ access to private community"
❌ Bad: Just main course image alone

#### Rule 2: BIG NUMBER prominent

The single biggest converter Maria found.

Examples:
- "100+ Templates"
- "365 Days of Content"
- "30 Healthy Recipes"
- "50+ Email Templates"
- "12 Modules + 8 Bonuses"

→ Make number HUGE, bright, contrast color (red/yellow against neutral background).

#### Rule 3: Visual + Value Words combo

Don't just show pretty image. Add value words:
- "Easy-to-follow video tutorials"
- "Track your progress weekly"
- "Mobile-friendly format"
- "Lifetime updates"

→ Image alone = pretty but vague. Image + value words = high conversion.

## 🎬 MOCKUP VIDEO (Sam Cart top placement)

### Why video > static image?

- Static mockup = good
- Video mockup = **3x better conversion** per Maria

### Video structure (15-30 sec)

```
0-3s: BIG PRODUCT NAME on screen + cinematic music
3-8s: Scroll through course interface (Kajabi)
8-15s: Show all bundled items with motion
15-22s: BIG NUMBER reveal ("100+ Templates")
22-28s: Value bullets quick scroll
28-30s: "Start now" CTA + price
```

### Tools to make mockup video

- **Canva Pro** — animated mockups templates
- **Capcut** — easy editing on phone
- **Apple Keynote** — animations + export to video
- **Maria's recommendation**: Canva Pro is enough

### Embed in Sam Cart

- Top of checkout page (above fold)
- Auto-play muted (so not annoying)
- Loop forever
- Mobile-optimized (vertical or square)

## 🔥 SAM CART CONVERSION BOOSTERS

### Booster 1: USE PROOF Popup

> Real-time "X just bought" notification

**How it works**:
- Notification pops up bottom-left every 30-60 sec
- "Sarah from Hanoi just purchased [Product Name]"
- Creates FOMO + social proof

**Setup**:
- Use Proof = third-party tool ($24-49/month)
- Connect to Sam Cart via API
- Customize messages

**Impact**: +10-20% conversion (Maria's tested data)

**VN adaptation**:
- "Chị Hương từ Hà Nội vừa mua [Product]"
- "Em Linh từ TP.HCM vừa join cohort"

→ Make sure NAMES feel real (don't fake)

### Booster 2: ADD BILLING INFO TO FORM

**Why**: Reduces card declined rate.

When customer enters billing address matching card → bank trust score up → less likely to decline.

**Setup in Sam Cart**:
- Settings → Checkout Form → Add billing fields
- Make required
- Auto-fill from PayPal/Apple Pay if possible

**Impact**: +5-10% completion rate (less abandoned at "card declined")

### Booster 3: ORDER BUMP at checkout

**What**: Add-on offer right at checkout (one click to add)

**Examples**:
- Main course $97 + Order bump "Add Hook Writer template for $7"
- Main course $97 + Order bump "Add 1-1 review for $27"

**Impact**: 30-50% of buyers add the bump → AOV (average order value) up 20-30%

**Maria's formula**:
- Order bump price = 5-15% of main product
- Order bump = direct extension/enhancement of main
- 1 click = no friction

### Booster 4: ONE-CLICK UPSELL post-purchase

**What**: After purchase confirmation, show ANOTHER offer (next product in ladder)

**Example flow**:
1. Buyer purchases $97 mid-tier
2. Confirmation page → "Wait! Add comprehensive bundle for $50 more (vs $97 separately)"
3. One click → already have payment info → frictionless

**Impact**: 20-30% take upsell → AOV +30-50%

**Maria's note**: Powerful BUT requires careful design — don't push too hard.

### Booster 5: PAYMENT PLANS for high tier

**For $200+ products**:
- Offer "Pay $97 today, $97 in 30 days, $97 in 60 days"
- Reduces sticker shock
- More buyers commit

**Sam Cart supports natively**.

## ⚠️ MOCKUP MISTAKES (Don't do)

### 1. Generic stock photo
❌ Random stock photo "people working on laptop"
✅ Custom mockup show ACTUAL product

### 2. Too many items, no hierarchy
❌ 20 items with same size, audience overwhelmed
✅ 1 BIG main item + 4-6 smaller bundled icons

### 3. No big number
❌ "Course about social media"
✅ "100+ Templates inside"

### 4. Watermark or "draft" text
❌ Mockup looks unfinished/sample
✅ Polished, looks like real product

### 5. Bad image quality
❌ Blurry / low-res / pixelated
✅ HD, sharp, professional

### 6. Inconsistent style
❌ Different fonts/colors between mockup and brand
✅ Match brand exactly

### 7. No "+" sign suggesting more
❌ Show only listed items
✅ "+ more bonuses inside" creates curiosity
