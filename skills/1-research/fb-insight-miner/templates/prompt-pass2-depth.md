# Pass 2 — Depth Analysis

Taking each of the 9 dimensions + Uncategorized bucket from Pass 1, go DEEP.

## For each dimension, produce:

### 2.1 Sub-theme detailed map
- Split the dimension into 3-5 sub-themes (refine Pass 1)
- For each sub-theme:
  - Count
  - Top 3 verbatim quotes with post context (post URL + likes it got)
  - **Emotional intensity** rating (1-5, where 5 = visceral/raw/anguished)

### 2.2 Language patterns
- **Repeat words** (>5 times): list them
- **Metaphors** they use (e.g., "cảm giác như ngộp thở")
- **Emoji clusters** that appear with this dimension
- **Sentence structure** patterns (câu cảm thán? câu hỏi? câu tuyên bố?)

### 2.3 Speaker profile hints
For each dimension, try to infer WHO is speaking:
- Gender (based on name suffix: "chị", "em", "anh"; pronouns; topics)
- Life stage (married? has kids? student? retired?)
- Experience level (newbie asking basics vs advanced asking nuance)
- Emotional state (hopeful vs desperate vs angry)

### 2.4 Copy ammunition
For each dimension, extract:
- **3 phrases** quotable directly in marketing copy
- **1 hook headline** inspired by this dimension (write it as if for an Instagram reel)
- **1 CTA** that would resonate with this emotional state

## Focus: verbatim quotes
- Minimum **10 verbatim quotes per dimension**
- Show post context for each (URL, post likes count, comment likes count)
- Flag quotes that are "💎 GOLD" (highly unique, visceral, specific, copy-able)
- NEVER translate or paraphrase
