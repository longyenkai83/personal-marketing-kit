# Workflow A — Build Full Launch (1-2 tuần)

> **End-to-end launch sản phẩm số đầu tiên**: idea → reels → email → tripwire → mid-tier sale
> 
> **Time**: 1-2 tuần (5-10h/tuần)
> 
> **Skills used**: 7/10 skills

---

## 🎯 Outcome target

Sau workflow này anh chị có:
- ✅ Validated product idea (3-Pillar pass)
- ✅ Lead magnet PDF live
- ✅ Welcome email sequence chạy auto
- ✅ Tripwire $1-25k flow setup
- ✅ Mid-tier $890k-2.3tr ready to sell
- ✅ 5 reel scripts ready-to-shoot
- ✅ 30 caption batch
- ✅ First 5-10 paying customers

## 📅 Day-by-Day Plan

### TUẦN 1: Validate + Build

#### Day 1-2: Validate idea (digital-product-builder)

```
Open Claude Code → gõ:

"Apply digital-product-builder cho idea [topic của anh chị].
Run 3-Pillar test:
- Pique Curiosity?
- Super Specific Problem?
- Promise Unbelievable?

Audience target: [persona cụ thể — vd: 'phụ nữ KD 30-45 muốn launch khoá đầu']
Product type: [course / pdf_bundle / audio / hybrid]
Pricing target: [890k / 2.3tr / etc]

Output: validation report + Go/No-Go verdict."
```

**Decision gate**: 
- ≥24/30 score = ✅ Proceed
- <24/30 = 🔴 Re-do topic (loop back)

#### Day 3-4: Build lead magnet (lead-magnet-factory)

```
"Apply lead-magnet-factory cho topic [chosen above].
Format: 9-page Marie Forleo listicle PDF + 2 audio bonus.
Bridge to mid-tier product: [product name].
Output: 9-page outline + Canva-ready CSV + 2 audio scripts."
```

→ Polish PDF trong Canva (4-6h work).
→ Output deliverable: `lead-magnet.pdf` ready.

#### Day 5: Setup tripwire (tripwire-builder)

```
"Apply tripwire-builder.
Lead magnet: [PDF từ Day 3-4].
Mid-tier product: [name + price 890k].
Tripwire price: 25k worksheet.
Output: tripwire content + Welcome Coupon flow + post-purchase sequence."
```

#### Day 6-7: Build email sequences (email-marketing-writer)

```
"Apply email-marketing-writer.

Build 2 automation sequences:

1. Welcome Coupon Flow (4 emails):
   - Email 1 (immediate): Deliver PDF + tripwire 25k offer
   - Email 2 (day 1): Story + soft mention coupon
   - Email 3 (day 2): Urgency 24h
   - Email 4 (day 3): Last chance

2. Post-Purchase Flow (5 emails over 7 days):
   - Email A (immediate): Welcome buyer + delivery
   - Email B (day 1): Encouragement to use
   - Email C (day 3): Story + bridge to mid-tier
   - Email D (day 5): Soft pitch mid-tier
   - Email E (day 7): Hard pitch + bonus stack

Subject lines: random + curiosity Maria-style
Tone: personal, look-personal layout
CTA: 1 per email, clear
Persona: [persona]"
```

```

```

```
Generate FB/IG captions accompanying each reel.

Hashtag: 3 broad + 5 niche + 2 branded per caption.

Output: 5 captions + hashtag sets."
```

## ⚠️ Common Pitfalls

### Pitfall 1: Skip 3-Pillar test
❌ "Em đã chắc chắn topic — skip validation"
→ Mất 7 ngày build, launch flop, lose momentum

✅ Always run 3-Pillar test (15 phút). Score ≥24/30 trước khi proceed.

### Pitfall 2: Perfectionism trên content
❌ Quay 10 takes mỗi reel, edit 4h mỗi cái
✅ 80% quality, ship fast, iterate dựa trên data

### Pitfall 3: Mọi reel = "Comment X nhận DM Y"
❌ Audience burnout sau reel #3
✅ Diversify CTA (xem `shared/references/cta-library.md`)

### Pitfall 4: Email layout marketing template
❌ Header logo + footer social → vào Promotions tab
✅ Plain personal layout (look-personal pattern)

### Pitfall 5: Pricing $300+ out of gate
❌ Charge premium vì "em xứng đáng" → flop
✅ Sweet spot 890k-2.3tr, raise after 100+ sales
